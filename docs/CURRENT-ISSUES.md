1. 确认 `window.__APP_BUILD__` 与本文一致。
2. 看 Network：失败 URL、状态码、响应类型、单图体积。
3. 看后台 `admin.html` 的运行监控和卡片库抽检。
4. 运行 `npm run check:predeploy` 与相关生产审计脚本。
5. 证明根因后再改；不要用清空 localStorage、反复刷新或全墙重排当最终修复。
