# 搜索引擎收录与品牌展示（小白版）

> 站点：https://prompt-hub.cn · 更新：2026-06-06

---

## 1. 国内是不是「都能搜到」了？

**不完全是。** 你提交的是**百度站长**，不等于搜狗/360/手机百度/App 内搜索立刻都有。

| 平台 | 说明 |
|------|------|
| **百度** | 提交后 PC 百度、部分合作入口较快；**收录 ≠ 每个关键词第一名** |
| **必应** | Edge 默认常用必应；索引与百度**分开**，会慢几天到几周 |
| **手机百度 App** | 与 PC 百度缓存不同，可能**更晚**才出现 |
| **Google** | 未提交则海外几乎搜不到；对国内用户影响小 |

---

## 2. 为什么只有 Edge 能搜到，手机搜不到？

这很常见，**多半不是网站坏了**。

| 原因 | 解释 |
|------|------|
| **搜索引擎不同** | 电脑 Edge 默认**必应**；手机常是**百度 App** 或自带搜索，索引库不一样 |
| **你提交的是百度** | 在 Edge 用必应搜到，不代表手机百度已有 |
| **个性化/历史** | Edge 可能根据你**访问过** prompt-hub.cn，在建议或结果里提前露出 |
| **关键词不同** | 「提示词仓库」「Prompt Hub」「prompt-hub.cn」结果差很多，请固定一个词对比 |
| **新站波动** | 1～4 周内不同设备结果不一致是正常的 |

**建议自测（无痕/隐私模式）：**

1. 电脑 Chrome 无痕：百度搜 `Prompt Hub 提示词仓库`、`site:prompt-hub.cn`  
2. 手机百度 App：同样两个词  
3. 电脑 Edge 无痕：必应同样两个词  

`site:prompt-hub.cn` 若**有结果** = 该引擎已收录；若**无** = 该引擎还没收录。

## 3. 为什么没图标（灰色地球）？

| 步骤 | 做法 |
|------|------|
| **第 1 步** | 部署后打开 https://prompt-hub.cn/favicon.ico ，必须 **200 且能看到小图** |
| **第 2 步** | 百度站长 https://ziyuan.baidu.com → 抓取诊断 → 首页 → **更新/重新抓取** |
| **第 3 步** | 等 **1～4 周**；图标比标题晚更新很正常 |
| **第 4 步** | 必应 Webmaster 同样提交 sitemap 并请求索引 |

---

## 4. 多条结果、知乎/GitHub 在下面？

- **同站 2 条**：引擎整理期，以后会合并。  
- **知乎 / GitHub**：别人内容或**同名仓库**，删不掉，靠内容和外链竞争。  
- **「其他用户还搜索过」**：百度相关搜索，无法关闭。  

**无法保证永远第一**；能做的是技术合规 + 持续更新 + 品牌词外链。

---

## 5. Google 提交 = 国外能搜到吗？

| 问题 | 答案 |
|------|------|
| 有什么用？ | 帮助 **Google 索引**，海外用户用 Google 时有机会看到 |
| 怎么做？ | **分步教程**：**`docs/GOOGLE-SEARCH-CONSOLE-BEGINNER.md`**（验证 DNS / 提交 sitemap / 请求编入索引） |
| 和百度冲突吗？ | **不冲突**，建议两个都做 |

---

## 6. 推荐操作顺序

1. `.\deploy-pages.ps1` 部署（含 `favicon.ico`）  
2. 验证 https://prompt-hub.cn/favicon.ico  
3. 百度站长提交 sitemap + 重新抓取首页  
4. （可选）必应 Webmaster + Google Search Console  
5. 等 2～4 周；发带官网链接教程，品牌名统一 **Prompt Hub 提示词仓库**
