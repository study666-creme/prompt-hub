# 浏览器插件（Prompt Hub 快捷存卡）

> 状态：**v1.0 已实现** · 目录 `extension/`

## 功能

| 能力 | 说明 |
|------|------|
| 悬浮面板 | 图片区 + 提示词 + 保存 / 小号清空 |
| 拖图自动存 | 拖入图片后自动调用 API 保存并清空 |
| 划词复制 | **仅用户划选文字**后出现「复制到提示词」按钮 |
| 登录 | 打开 prompt-hub.cn 登录，bridge 脚本同步 session |

## 合规设计

- 首次使用需确认免责声明
- 不自动扫描 DOM、不批量抓取
- 划选长度上限 8000 字
- 面板底部常驻法律提示

## 安装

见 [`extension/README.md`](../extension/README.md)。

## 服务端

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/extension/status` | 登录态 |
| POST | `/api/v1/extension/quick-card` | 追加一张卡片 + 可选图片上传 |

部署 Worker 后生效；CORS 已允许 `chrome-extension://`。
