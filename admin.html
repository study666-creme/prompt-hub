图片存储（R2）

卡片库「裂图 / 502 / 404」多为 R2 未同步或旧第三方外链失效。本机已有 backups/card-images 时，在项目根目录 PowerShell 执行：

node scripts/sync-supabase-to-r2.mjs --from-local --out backups/card-images --skip-existing

需先在 scripts/admin.local.env 填 R2 密钥。仍可从 Supabase 拉取并上传：node scripts/sync-supabase-to-r2.mjs --out backups/card-images --skip-existing
