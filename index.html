让你在任何页面光速保存图片和文字到 Prompt Hub 卡片库。

下载浏览器插件（zip）

Chrome / Edge 通用 · 约 200KB · 下载后解压到任意文件夹

安装

  1. 点击上方按钮下载 zip，解压到电脑。
  2. 打开 Chrome 浏览器或 Edge 浏览器 → 地址栏输入 chrome://extensions（Edge 为 edge://extensions）。
  3. 打开右上角开发者模式 → 点加载已解压的扩展程序 → 选中刚解压的文件夹。
  4. 打开 prompt-hub.cn 并登录，扩展会自动同步账号。

怎么用

  1. 点击浏览器右上角 Prompt Hub 存卡 图标 → 在当前页打开面板（首次需允许访问当前网站）。
  2. 鼠标悬停段落 → 点「复制到提示词」；或划选文字 →「复制到提示词」。
  3. 拖入 / 粘贴图片（无需选中图片框，截图完直接 Ctrl+V 即可）到面板。
  4. 填入图片和文字后卡片自动保存到仓库（可在面板 ⚙ 设置里关闭）。

小提示

  • 面板 × 关闭本页；− 只是收起，内容不会丢。
  • 只保存您手动选择 / 拖入 / 粘贴的内容，不会自动抓取整页。
  • 保存后回到卡片库刷新即可看到新卡片（已登录会自动云同步）。
