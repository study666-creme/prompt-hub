本文件只记录用户可感知或架构级变化；完整细节以 Git 提交历史为准。

## 2026-07-11

- 生图目录收敛为卡藏 API 的全能模型2/香蕉与 Apimart MJ，卡藏价格按 `1 元 = 100 积分`实时换算。
- 积分充值取消折扣和赠送会员天数，所有积分包固定按 `1 元 = 100 积分`定价。
- 精简运营后台旧供应商、重复社区维护和冒烟入口；存储巡检改为按需扫描当前 R2/MemFire 主存储。
- 修复后台错误检测旧生图密钥、账号删除遗漏 R2 对象和旧模型 ID 回退问题。
- 清理过时补丁、测试页、缓存文件和重复交接文档，建立统一文档索引。
- 将主站 canonical、Open Graph、sitemap、robots 和扩展登录链路统一到 `prompt-hubs.com`。
- 新增文档断链检查与 GitHub Actions CI。
- 更新隐私政策中的 MemFire、R2 和浏览器扩展数据说明。
